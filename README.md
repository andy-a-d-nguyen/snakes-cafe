### Setup

Make sure you are in the repo's root directory first and that you have python 3.10 installed.

Create a `venv` environment:

```bash
python3 -m venv .venv
```

Activate `venv` environment:

```bash
source .venv/bin/activate
```

Deactivate `venv` environment:

```bash
deactivate
```
